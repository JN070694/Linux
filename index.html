Linux Mint
Scratch Setup

This is the preferences setup page, predominantly used for Linux Mint.

General:

  • Add firefox extensions, like uBlock Origin.
  • Right click on the panel and click expand options, add "show desktop feature" move it to far right.
  • Power Manager: The "System" tab needs to be at suspend for 15 mins and the "Display" tab needs to be deactivated as it doesnt work with all hardware.
  • These power manager settings may be the default.

Software:

  • Bitwarden
  • Discord
  • FreeTube
  • HDtodayz
  • Heroic Games Launcher
  • Inhibit, AKA "Keep Me Awake"
  • OBS Studio
  • PDFsam
  • Used to merge PDF documents.
  • Steam
  • Once controller is setup, go to Steam settings > controller > and set "guide button focuses Steam" to off.

Special Software:

  • ProtonVPN: This is a free VPN that has to be downloaded via the terminal. These are the commands, entered one line at a time:
  • wget https://repo.protonvpn.com/debian/dists/stable/main/binary-all/protonvpn-stable-release_1.0.8_all.deb
  • sudo dpkg -i ./protonvpn-stable-release_1.0.8_all.deb && sudo apt update
  • sudo apt install proton-vpn-gnome-desktop
  • ProtonVPN Download Instructions

Gaming Settings:

  • Driver Manager: The default, open-source, driver (Nouveau) is not compatible for gaming.
  • The latest recommended driver in Driver Manager should be equipped and the computer restarted.
  • Secure Boot: The "Secure Boot" feature in the BIOS blocks Nvidia drivers from being accessed.
  • For Alienware products, you need to press the F2 key on startup to get to the BIOS settings.
  • In the security tab, you need to go to Secure Boot settings and deactivate this feature.
  • Audio: After you update your driver and disable Secure Boot, you may not have audio in your games.
  • apt purge pipewire pipewire-bin
  • systemctl enable --user pulseaudio
  • sudo reboot
  • PulseAudio Restore
  • Controller: After your audio is working, you can set up the controller via bluetooth.
  • Option 1: xone for Dongle Pairing
  • xone, see Guide instructions
  • Option 2: xpadneo for Bluetooth Pairing
  • xpadneo, see Installer terminal commands

Launchers:

  • Heroic Game Launcher:
  • Download the game, right click, settings, select proton-latest
  • Steam:
  • Download the game, right click, settings, select Proton-hotfix
